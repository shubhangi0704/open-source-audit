# Open Source Audit – Linux Shell Scripts

This repository contains five basic shell scripts demonstrating practical Linux command-line skills. These scripts reflect the philosophy of open-source software — transparency, automation, and user control.

## Scripts Overview

### 1. script1_welcome.sh
Displays system information like Linux distribution, kernel version, user details, uptime, and license information.

### 2. script2_file_organizer.sh
Automatically organizes files in a directory into folders based on file type.

### 3. script3_system_monitor.sh
Shows system performance metrics such as CPU usage, memory usage, and disk space.

### 4. script4_backup.sh
Creates a backup of a specified directory into a compressed archive.

### 5. script5_process_checker.sh
Checks if a given process is running and displays its details.

## Requirements
- Linux-based OS
- Bash shell
